# 🥛 Amul Animated Website

An interactive and visually engaging animated website inspired by Amul's iconic branding. The website features smooth animations, modern UI design, responsive layouts, and an immersive user experience showcasing Amul products.

## 🚀 Features

- ✨ Smooth animations and transitions
- 📱 Fully responsive design
- 🎨 Modern and attractive UI
- 🥛 Amul-themed product showcase
- ⚡ Fast loading and optimized performance
- 🌐 Cross-browser compatibility
- 🎯 Interactive user experience

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- GSAP (Animations)
- Tailwind CSS (if used)
- React.js (if used)
